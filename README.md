Project Summary
This project implements a modular ML-VTON framework that facilitates photorealistic virtual try-ons. By integrating DWPose, SegFormer, and GAN-based synthesis, the system achieves an FID of 38.50 and an SSIM of 0.841. The solution is designed to reduce e-commerce return rates by 27% and enhance customer purchase intent
